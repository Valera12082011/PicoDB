**PicoDB** — это сверхлегкая in-memory база данных, написанная на современном C++, которая объединяет простоту Redis с мощью реактивных вычислений. Проект спроектирован с упором на радикальный минимализм, читаемость и отсутствие излишних абстракций.

## ⚡ Преимущества PicoDB

* **Минимализм и легкость:** Чистая кодовая база без перегруженной архитектуры. Проект легко понять "с нуля" и модифицировать под свои нужды.
* **Реактивность (BOND):** База не просто хранит данные, она умеет их вычислять. Связывайте ключи формулами, и PicoDB автоматически обновит результат при изменении базовых значений.
* **Zero-Copy & Move Semantics:** Высокая производительность благодаря использованию современных возможностей C++ для исключения лишнего копирования данных.
* **Встроенный MathEngine:** Собственный рекурсивный парсер математических выражений без использования тяжелых сторонних библиотек.
* **Сжатие Zstd:** Эффективная бинарная сериализация с использованием алгоритмов Zstandard для экономии места при сохранении слепка памяти на диск.

## 🚀 Быстрый старт (PiQ Query Language)

PicoDB использует простой интуитивный текстовый протокол. Главная киллер-фича — команда `BOND`, которая задает математическую связь между ключами:

```text
> SET price 100
> SET tax 0.2
> BOND total = price + (price * tax)
> GET total
"120.000000"

> SET price 200
> GET total
"240.000000"
```

🔧 Сборка и запуск
# 1. Клонирование репозитория
git clone [https://github.com/ТВОЙ_НИК/PicoDB.git](https://github.com/ТВОЙ_НИК/PicoDB.git)
cd PicoDB

# 2. Подготовка и сборка
mkdir build && cd build
cmake ..
cmake --build .

# 3. Запуск базы данных
./picodb
