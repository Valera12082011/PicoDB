
# PicoDB — Minimalist C++ Reactive Database

<img width="1024" height="559" alt="image" src="https://github.com/user-attachments/assets/211ff22f-13ed-4250-9f4a-40bd70627952" />

PicoDB — это сверхлегкая in-memory база данных, написанная на современном C++ и Cmake, которая объединяет простоту Redis с мощью реактивных вычислений.

## ⚡ Почему PicoDB?

* **Микро-архитектура:** Вся логика в 5 файлах. Читаемый, понятный и расширяемый код.
* **Реактивность (BIND):** База не просто хранит данные, она умеет их вычислять. Связывайте ключи формулами, и PicoDB сама обновит результат при изменении базовых значений.
* **Zero-Copy & Move Semantics:** Максимальная производительность за счет жесткого запрета на лишнее копирование данных.
* **Встроенный MathEngine:** Собственный парсер математических выражений без тяжелых внешних зависимостей.
* **Сжатие Zstd:** Быстрая бинарная сериализация с использованием алгоритмов Facebook Zstandard.

## 🚀 Быстрый старт (PiQ Query Language)

PicoDB использует интуитивно понятный текстовый протокол PiQ:

> SET price 100
> SET tax 0.2
> BIND total = price + (price * tax)
> GET total
"120.000000"
> SET price 200
> GET total
"240.000000"

## 🔧 Сборка
PicoDB использует CMake для кроссплатформенной сборки. Убедитесь, что у вас установлен компилятор с поддержкой C++17/20 и библиотека Zstandard (`libzstd`).

# Клонируем репозиторий
git clone [https://github.com/ТВОЙ_НИК/PicoDB.git](https://github.com/ТВОЙ_НИК/PicoDB.git)
cd PicoDB

# Создаем папку для сборки
mkdir build && cd build

# Конфигурируем и собираем проект
cmake ..
cmake --build .

# Запускаем базу!
./picodb
